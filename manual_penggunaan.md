Manual Penggunaan Sistem E-KOST

Selamat datang di E-KOST, platform yang menghubungkan pencari kost dengan pemilik kost. Dokumen ini berisi panduan lengkap penggunaan sistem untuk semua tipe pengguna.

Tipe Pengguna (Role)

Sistem ini memiliki 3 tipe pengguna:

1. Pencari Kost (Mahasiswa/Umum): Mencari, melihat detail, dan menyimpan kost favorit.
2. Pemilik Kost (Owner): Mengelola data kost, fasilitas, dan gambar.
3. Admin: Memverifikasi data kost dan mengelola feedback.

Memulai (Registrasi & Login)

Semua pengguna (kecuali Admin yang dibuat lewat database) dapat mendaftar melalui halaman registrasi.

1. Buka halaman Register.
2. Isi Nama, Email, No HP, dan Password.
3. Pilih Peran (Role):
    - Pilih Mahasiswa jika Anda ingin mencari kost.
    - Pilih Pemilik jika Anda ingin menyewakan kost.
4. Klik Register.
5. Setelah berhasil, Anda akan diarahkan ke halaman utama sesuai peran Anda.

Panduan untuk Pencari Kost (Mahasiswa)

Setelah login sebagai Mahasiswa/Pencari, Anda akan diarahkan ke Halaman Utama (Home).

1. Mencari Kost

-   Di Halaman Utama, Anda dapat melihat daftar rekomendasi kost.
-   Gunakan fitur Cari (jika tersedia di beranda) atau navigasi ke halaman kost untuk memfilter berdasarkan lokasi atau harga.

2. Melihat Detail Kost

-   Klik pada kartu kost untuk melihat halaman Detail Kost.
-   Informasi yang tersedia:
    -   Nama & Deskripsi Kost
    -   Harga per bulan/tahun
    -   Fasilitas (AC, WiFi, dll)
    -   Galeri Foto
    -   Lokasi

3. Menyimpan Favorit

-   Pada halaman Detail Kost, klik tombol Favorit.
-   Kost akan tersimpan di daftar favorit Anda.
-   Untuk melihat daftar favorit, akses menu Favorit dari navigasi profil.

4. Memberikan Komentar

-   Anda dapat memberikan pertanyaan atau ulasan di kolom komentar pada halaman Detail Kost.

Panduan untuk Pemilik Kost (Owner)

Setelah login sebagai Pemilik, Anda akan diarahkan ke Dashboard Owner.

1. Menambah Kost Baru

-   Di Dashboard, klik tombol Tambah Kost.
-   Isi formulir data kost secara lengkap:
    -   Nama Kost
    -   Deskripsi
    -   Harga
    -   Lokasi/Alamat
    -   Tipe (Putra/Putri/Campur)
-   Simpan data. Kost baru akan berstatus Pending menunggu verifikasi Admin.

2. Mengelola Foto Kost

-   Pilih kost yang ingin diedit.
-   Masuk ke menu Kelola Gambar.
-   Unggah foto-foto terbaik kost Anda.
-   Anda juga dapat menghapus foto yang sudah tidak relevan.

3. Mengatur Fasilitas

-   Pastikan Anda mencentang fasilitas yang tersedia (misal: WiFi, Parkir Motor, Kamar Mandi Dalam) agar kost Anda lebih menarik bagi pencari.

Panduan untuk Admin

Admin memiliki akses ke Dashboard Admin.

1. Verifikasi Kost

-   Admin bertugas memeriksa data kost baru yang didaftarkan oleh Owner.
-   Buka menu Verifikasi Kost.
-   Tinjau data kost.
-   Klik Approve (Setujui) untuk menayangkan kost, atau Reject (Tolak) jika data tidak sesuai.
-   Kost yang disetujui akan muncul di halaman pencarian publik.

2. Manajemen Feedback

-   Admin dapat melihat masukan dari pengguna melalui menu Feedback.
-   Data feedback dapat diexport ke format CSV untuk pelaporan.

Bantuan & Feedback

Jika Anda mengalami kendala atau memiliki saran:

-   Akses halaman Bantuan di footer situs.
-   Gunakan formulir Feedback untuk mengirimkan pesan langsung kepada admin.
